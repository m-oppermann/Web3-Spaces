### About

This repository contains a simple full-stack application that utilizes web3 technology, enabling users to securely and anonymously share thoughts, explore new ideas and reward each other.

### Stack

- Nextjs
- Typescript
- Tailwindcss
- Radix
- Heroicons
- Framer Motion
- Wagmi (w/ Viem)
- Prisma
- Planetscale
- Vercel
